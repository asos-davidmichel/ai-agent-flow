🚀 Agile Flow Metrics Dashboard

📈 Throughput Trend

Shows the number of items completed over time.

⏱️ Cycle Time by Completion Date

Shows how long items took to complete (from start to done), measured in calendar days.

🕐 Lead Time by Completion Date

Shows the total time from creation to completion for each item.

⚡ Flow Efficiency (Cycle / Lead Time)

Shows the percentage of time spent in active work compared to total lead time.

🐛 Bug Rate Trend

Shows the percentage of bugs relative to all completed work items over time.

🔄 Cumulative Flow Diagram

Shows arrival vs departure as cumulative lines with trend lines to visualize system stability. Toggle states on/off to analyze specific workflow stages.

Arrival Rate: - items/week | Departure Rate: - items/week

🔄 Transition Rate Ratios

Shows the arrival-to-departure rate ratio for each workflow transition.

⚖️ Sustainability (Net Flow)

Shows the difference between work started and work completed each day.

⏱️ Time In Board Column

Total amount of time (in calendar days) items spent in a given column.

📊 Backlog State Distribution

Shows how work items are distributed across different workflow states.

📈 Daily Work In Progress (WIP)

The number of items in progress each day.
A red dot represents when a column breached its WIP limit on that date.

📊 Daily WIP x Work Item Age

Maintain a focus on WIP whilst seeing when work item age grows.

🔧 Aging Work In Progress

Shows currently active items grouped by how long they've been in progress.

📍 Work Item Age

How many days an individual work item (represented by a dot) has been in-progress for.

🚫 Blocked Items Over Time

Shows the number of blocked items each day with a trend line.

⚠️ Stale Work

Shows how many days since a work item has been updated.
